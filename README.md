# SnakeGame

A little game written in C++ about a snake that eats apples.  
There are no textures in game just squares and circles.

## Requirements
|Tool|Version|
|--|--|
|CMake|3.8+|
|Visual Studio|2022|
|C++|14|

## How to build
1. Run `build.bat`
2. Open the folder in Visual Studio
3. Run the solution

## Controls
* WASD - movement
* Space - restart
* ESC - quit the game
